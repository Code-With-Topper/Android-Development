# Android Development Practicals

This repository contains a collection of Android practice projects organized under `Android_Practicals/`.

## Clone the repository

Use the repository URL to clone the project locally:

```bash
git clone https://github.com/Code-With-Topper/Android-Development.git
cd Android-Development
```

If you are using SSH and have access to the repository, use:

```bash
git clone git@github.com:Code-With-Topper/Android-Development.git
cd Android-Development
```

## Open and run

1. Open Android Studio.
2. Choose "Open" and select the `Android_Practicals/` folder.
3. If a project folder is not recognized as a full Gradle project, create a new Android Studio project and copy the XML and Java files from the exercise folder into the generated `app/src/main/` structure.
4. Run the app on an emulator or physical device.

## Notes

- Some practice folders may not include a full Android Studio/Gradle project structure.
- The code is intended for learning Android concepts, so some examples may be simplified.
- Use each folder as a reference or copy files into a new project for testing.

## Contribution

If you want to add new exercises or improve existing examples:

1. Create a new folder under `Android_Practicals/`.
2. Add activity XML and Java files.
3. Commit your changes and push them to the repository.
